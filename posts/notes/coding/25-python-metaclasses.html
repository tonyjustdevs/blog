1. Introduction

1.1 What is a metaclass?

A metaclass is a class that defines the behavior of the classes and their instances:

  • A metaclass is a class whose instances are classes.

1.2 Why?

metaclass allows for dynamically (i.e. run time) customize the creation of classes:

  • A metaclass can change the behavior of a class at run time.

2. type Base Class

type is base (default metaclass) of all objects in Python

3. Creating Classes with type:

The type function can dynamically (ie run-time) create classes

type becomes a class constructor when called with 3 arguments

type(name, bases, dict) -> a new type

  • name: String name of the new class.
  • bases: Tuple of parent classes (inheritance).
  • dict: Dictionary of attributes and methods for the class.

# Define a simple metaclass
class MyMeta(type):
    # The __new__ method is called when a class is created
    def __new__(cls, name, bases, dct):
        print(f"Creating class {name} with MyMeta")
        return super().__new__(cls, name, bases, dct)

# Create a class using the metaclass
class MyClass(metaclass=MyMeta):
    pass

# Instantiate the class
obj = MyClass()
